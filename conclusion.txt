1. Que se aprendio durante el desarrollo del proyecto:
Durante el desarrollo del proyecto se adquirieron disntintas bases fundamentales sobre el manejo de datos en archivos estadísticos, empleando las clases como BufferedReader y FileReader. Igualmente, se logro manipular y centralizar los datos en diferentes tipos de archivos obteniendo coherencia en los resultados y una ejecución limpia. De igual forma, se estableció una estructura centralizada para clases, métodos, entidades y utilidades para desarrollar la capacidad de adaptación y respuesta del sistema. Por ultimo, se estructuró un entorno de control de versiones, para que varios usuarios puedan trabajar simultáneamente.

2. Posibles aplicaciones de lo aprendido a nivel profesional:
Algunas aplicaciones de las que podemos sacar partido a nivel profesional del conocimiento adquirido al desarrollar el proyecto están encaminadas inicialmente al trabajo en equipo, escalado de entornos de desarrollo agil, uso de estrategias para garantizar un trabajo en grupo eficiente y proactivo, y finalmente el generar estrategias basandonos en diferentes conceptos fundamentales para el manejo de datos en diferentes tipos de archivos como por ejemplo CSV, TXT, entre otros.

3. Dificultades presentadas durante el proyecto
Algunas de las dificultades a las que nos enfrentamos durante el desarrollo de este proyecto se debieron a que inicialmente los textos no se generaban en una sola carpeta. También, se presento un error el cual impedia la separación correcta en algunos datos específicos. Hubo complicaciones en la manipulación de datos ya que no había una estructura cláramente definida con las clases y tampoco decidimos adpotar un patrón de diseño al cual apegarnos al momento de estructurar el proyecto, por lo que se debió adaptar el código existente a un patrón especifico (Patron de Servicios) para lograr una estructura escalable y legíble a futuro.
